Exercício: Manipulação de Notas de Alunos
Descrição
Você deve criar um programa que manipule um array de objetos representando alunos e suas notas. O programa deve realizar as seguintes tarefas:

Definir um array de objetos: Cada objeto deve ter as propriedades nome (string) e notas (array de números).

Função para calcular a média: Crie uma função chamada calcularMedia que recebe um array de notas e retorna a média dessas notas.

Função para encontrar alunos aprovados: Crie uma função chamada alunosAprovados que recebe o array de alunos e uma nota mínima para aprovação (por exemplo, 7). Essa função deve retornar um novo array contendo os nomes dos alunos que têm média igual ou superior à nota mínima.

Função para listar alunos e suas médias: Crie uma função chamada listarAlunos que exibe o nome de cada aluno e sua média.
